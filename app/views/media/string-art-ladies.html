String Art Ladies

String Art Ladies to cztery młode, piękne, utalentowane kobiety, absolwentki Akademii Muzycznej. Pomysłowo przekształcają najpopularniejsze utwory muzyki rozrywkowej z pięknem i intensywnością klasycznego instrumentarium smyczkowego. Nazwa zespołu pochodzi od sztuki String Art, która ukazuje różnorodne połączenia kolorystyczne nici. Podobnie jak kolorystyka String Art, repertuar kwartetu jest bardzo zróżnicowany. Poza standardami klasycznymi dziewczyny grają przede wszystkim muzykę rozrywkową. Na ich koncertach usłyszeć można utwory w stylu irlandzkim, gruzińskim, country, a także POP, ROCK, R&B. Koncerty kwartetu wykonywane są z towarzyszeniem znakomitego perkusisty Łukasza Łapińskiego.
